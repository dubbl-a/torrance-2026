# torrance-2026

This repository exists to preserve legacy URLs from the original Torrance 2026 campaign finance site.

The original site was renamed to [torrance-watch](https://github.com/dubbl-a/torrance-watch) and migrated to [torrancewatch.org](https://torrancewatch.org).

This repo serves only a single redirect page that maps legacy URLs to their new equivalents on torrancewatch.org:

| Legacy URL | New URL |
|---|---|
| /index.html | /finance/2026/mayor-2026/ |
| /kaji_2026.html | /finance/2026/district-1-2026/ |
| /mattucci_2026.html | /finance/2026/treasurer-2026/ |
| any other path | / (homepage) |

No further development happens here. All active work lives in the torrance-watch repo.
